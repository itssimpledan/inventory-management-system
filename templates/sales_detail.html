COGS & Inventory Costing

Atome is a payment gateway — its transaction report records how customers paid, not what was sold. Each row has an order ID but no SKU or quantity data.

To calculate COGS correctly, upload the corresponding order report from the originating sales platform (e.g. Shopify, Lazada, Shopee) for the same period. That report contains the SKU breakdown per order, which the system uses to pull FIFO unit costs and compute cost of sales.

Future enhancement: cross-reference Atome's "E-commerce Platform Order ID" against Shopify order exports to auto-link transactions to SKUs.
